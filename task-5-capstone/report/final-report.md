# 📊 Capstone Project Report

## Executive Summary
A penetration test was conducted on DVWA to identify vulnerabilities.

## Methodology
Recon → Exploitation → Analysis

## Findings
- SQL Injection (High)
- XSS (Medium)
- CSRF (Medium)

## Incident Response
- Detection: Suspicious queries
- Containment: Blocked access
- Eradication: Fixed vulnerability
- Recovery: System restored

## Recommendations
- Use prepared statements
- Implement CSP
- Use CSRF protection

## Conclusion
System is vulnerable but can be secured with proper measures.
