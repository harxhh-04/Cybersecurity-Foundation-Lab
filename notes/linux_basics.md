# Linux Basics

Linux is widely used in cybersecurity because it provides powerful command line tools.

## File Navigation

pwd – shows current directory  
ls – lists files  
cd – change directory  

Example:
cd Desktop
mkdir test
cd test

## File Creation
touch file.txt

## File Permissions
chmod 777 file.txt


Permission types:
- Read
- Write
- Execute

## Networking Commands
ifconfig
ping
netstat
traceroute


These commands help analyze network connectivity.
