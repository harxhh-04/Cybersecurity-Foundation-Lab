# Networking Basics

## OSI Model

The OSI model has 7 layers:

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

## TCP vs UDP

TCP
- Reliable
- Connection based

UDP
- Faster
- No connection required

## Common Ports

21 – FTP  
22 – SSH  
80 – HTTP  
443 – HTTPS  

## Network Scanning

Tools like Nmap are used to discover open ports and services.

Example:
nmap -sV <target_ip>


This command detects service versions running on open ports.
